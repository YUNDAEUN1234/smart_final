# 스마트제조 공정분석 대시보드 — 발표 스크립트 (5분)

## 1. 인트로 (30초)

안녕하세요. 스마트제조 공정분석 대시보드를 발표하겠습니다.

이 프로젝트는 제조 현장의 공정 데이터를 실시간으로 분석하고, 공정능력지수와 통계적 공정관리 결과를 한 화면에서 확인할 수 있도록 구현한 웹 기반 대시보드입니다.

파이썬의 스트림릿(Streamlit) 프레임워크를 활용하여 개발했으며, 별도의 프로그램 설치 없이 웹 브라우저에서 바로 사용할 수 있습니다.

---

## 2. 데이터 입력 탭 (40초)

【화면: 📂 데이터 입력 탭】

먼저 데이터 입력 기능입니다.

데이터는 총 세 가지 방식으로 입력할 수 있습니다.

첫 번째는 샘플 데이터 자동 생성 기능입니다.

목표값, 허용오차, 부분군 개수와 크기 등을 설정한 뒤 데이터를 생성하면 정규분포를 기반으로 한 시뮬레이션 데이터가 생성됩니다.

두 번째는 CSV 또는 엑셀 파일 업로드 방식입니다.

【시연: sample_bearing_diameter.csv 업로드】

파일을 업로드하면 측정값 컬럼의 통계를 자동으로 분석하여 평균 ± 3시그마 기반의 규격 한계를 추천합니다. 이를 통해 사용자가 데이터에 맞지 않는 규격을 입력하는 실수를 방지할 수 있습니다.

세 번째는 데이터 편집기를 통해 값을 직접 수정하거나 입력하는 방식입니다.

## 3. 공정능력분석 탭 (1분 30초)

【화면: 📊 공정능력분석 탭】

다음은 핵심 기능인 공정능력분석 화면입니다.

분석은 크게 네 단계로 구성됩니다.

### 정규성 검정

먼저 샤피로-윌크 검정을 수행합니다.

여기서 유의확률, 즉 p값이 0.05 이상이면 데이터가 정규분포를 따른다고 판단합니다.

오른쪽의 분위수-분위수 그래프를 통해 시각적으로도 정규성을 확인할 수 있습니다.

만약 정규성을 만족하지 못하는 경우에는 박스-콕스 변환을 자동으로 제안합니다.

체크박스를 선택하면 최적의 람다 값을 적용하여 데이터를 변환하고, 변환된 데이터에 대해 다시 정규성 검정을 수행합니다.

### 분포 시각화

다음으로 상자그림과 히스토그램을 통해 데이터 분포를 확인할 수 있습니다.

그래프의 축 범위는 입력된 데이터의 분포에 맞춰 자동으로 조정되며, 규격 한계가 데이터 범위 안에 있으면 빨간 점선으로 표시됩니다. 만약 규격이 데이터 범위 밖에 있을 경우에는 화살표와 함께 위치를 안내하여, 어떤 규격의 데이터가 들어오더라도 그래프가 깨지지 않도록 설계하였습니다.

또한 데이터 평균이 규격 범위 밖에 있으면 경고 메시지를 표시하여 사용자가 규격 설정 오류를 빠르게 인지할 수 있습니다.

### 공정능력지수

세 번째는 공정능력지수 분석입니다.

여기서는 Cp, Cpk, Pp, Ppk 네 가지 지표를 제공합니다.

- Cp는 규격 대비 공정의 산포 수준을 나타내고
- Cpk는 산포와 함께 공정 중심의 치우침까지 반영합니다.
- Pp와 Ppk는 장기적인 공정 성능을 평가하는 지표입니다.

일반적으로 1.67 이상은 매우 우수, 1.33 이상은 우수, 1.0 미만은 개선이 필요한 수준으로 판단합니다.

### 종합 그래프

마지막으로 히스토그램과 정규분포 적합 곡선, 규격 한계선, 평균선, 공정능력지수를 하나의 그래프로 통합하여 제공합니다. 데이터의 분포 형태와 규격 대비 위치를 직관적으로 확인할 수 있습니다.

## 4. 통계적 공정관리 탭 (1분 20초)

【화면: 📈 통계적 공정관리 탭】

이 탭에서는 슈하트 관리도를 제공합니다.

데이터 특성에 따라 적절한 관리도를 자동으로 추천합니다.

- 계량형 데이터는 부분군 크기에 따라 Xbar-R, Xbar-S 또는 I-MR 관리도
- 계수형 데이터는 NP, P, C, U 관리도를 제공합니다.

【관리도 확인】

위쪽은 Xbar 관리도, 아래쪽은 R 관리도입니다.

초록 점선은 중심선, 분홍 점선은 관리 상한선, 빨간 점선은 관리 하한선을 의미합니다.

또한 빨간 점으로 표시된 부분은 관리 한계를 벗어난 이상 부분군입니다.

이상 탐지는 다음 세 가지 규칙을 적용하였습니다.

1. 관리 한계선 초과
2. 넬슨 규칙 2번 — 연속 9개 점이 중심선의 같은 쪽에 위치하는 경우
3. 넬슨 규칙 3번 — 연속 6개 점이 증가하거나 감소하는 추세를 보이는 경우

【2단계 확인】

이상 부분군이 발견되면 자동으로 2단계 분석이 활성화됩니다.

해당 부분군을 제외한 뒤 관리도를 다시 계산하여 보다 안정된 공정 상태에서의 관리 한계를 확인할 수 있습니다.

## 5. 종합 대시보드 탭 (40초)

【화면: 🖥️ 종합 대시보드 탭】

마지막은 종합 대시보드 화면입니다.

상단에는 핵심 성과 지표 형태로 평균값과 Cp, Cpk, Pp, Ppk를 요약하여 제공합니다.

아래에는 공정능력분석 그래프와 관리도를 좌우로 배치하여 공정의 전반적인 상태를 한 화면에서 확인할 수 있도록 구성하였습니다.

또한 하단의 종합 판정 영역에서는 공정능력 등급과 통계적 공정관리 상태를 색상으로 표시하여 관리자가 현재 공정 상태를 빠르게 판단할 수 있도록 하였습니다.
